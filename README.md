# Trash Catcher

Trash Catcher is a Unity serious game prototype focused on recycling awareness.  
The player controls a recycling bin and switches between waste categories to collect the correct trash while avoiding hazardous waste.

## My Role

Gameplay Programmer / Game Developer

## Project Overview

This project was developed as a university serious game demo.  
The main goal is to teach players the difference between recyclable waste categories through simple, fast-paced gameplay.

Players must collect the correct falling trash items by switching the active bin category. Hazardous waste creates an additional challenge because it cannot be selected as a bin category and must be avoided.

## Features

- Category-based bin switching system
- Falling trash spawning system
- Scoring and penalty logic
- Hazardous waste mechanic
- Level progression
- Object pooling for performance optimization
- Save/load structure
- Simple UI feedback for gameplay clarity

## Controls

- A / D or Left / Right Arrow: Move
- 1: Plastic
- 2: Paper
- 3: Glass
